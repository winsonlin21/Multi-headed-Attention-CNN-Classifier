# Multi-headed Attention CNN Classifier (MNIST 0–9)

A lightweight convolutional neural network with multi-headed attention that classifies handwritten digits (0–9). Achieves **~99% test accuracy** on MNIST.

---

## Features
- ✅ CNN backbone + multi-headed attention module
- ✅ ~99% test accuracy on MNIST
- ✅ Single-image interactive inference
- ✅ Simple training script and clean repo layout
