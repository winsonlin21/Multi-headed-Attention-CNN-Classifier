# Multi-headed Attention CNN Classifier (MNIST 0–9)

A lightweight convolutional neural network with multi-headed attention that classifies handwritten digits (0–9). Achieves **~99% test accuracy** on MNIST.

---

## Features
-  CNN backbone + multi-headed attention module
-  ~99% test accuracy on MNIST
- Easy to adapt to other datasets — tweak image size, channels, and class count


## Requirements
Install PyTorch package

## How to run
1) ```python run_inference.py```
2) You’ll be prompted to enter the path to an image containing a single digit.
3) Hopefully the predicition is correct.
